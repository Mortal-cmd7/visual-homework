# Visual Homework – YOLO Pose Tracking + Distance Estimation

## 功能简介

该程序基于 YOLO11 姿态估计模型和 ByteTrack 跟踪器，实现以下核心功能：

1. **自动识别与跟踪**  
   自动检测画面中的对象并分配唯一的跟踪 ID，实时绘制边界框。

2. **人体姿态与轨迹可视化**  
   鼠标点击选中行人后，显示其人体骨架（COCO 17 关键点）以及近 30 帧的运动轨迹。

3. **单目距离估算**  
   支持通过标定参考物估算选中目标到摄像头的实际距离（使用前需按 **K** 键进行人工标定）。

---

## 运行方式

### 基本命令

```python
python main.py [参数]
```

### 命令行参数

| 参数                    | 说明                                                       | 默认值                  |
| :---------------------- | :--------------------------------------------------------- | :---------------------- |
| `--source`              | 视频源：`0` 表示摄像头，也可传入视频文件路径或 RTSP 流      | `"0"`                   |
| `--view-img` / `--no-view-img` | 是否显示实时预览窗口                                       | 默认显示                |
| `--save-video`          | 是否保存输出视频                                           | 默认不保存              |
| `--output`              | 输出视频保存路径（需配合 `--save-video` 使用）             | `"tracking_output.avi"` |
| `--conf`                | 置信度阈值                                                 | `0.3`                   |
| `--iou`                 | IoU 阈值                                                   | `0.3`                   |
| `--device`              | 推理设备（如 `cpu`, `cuda:0`, `mps`）                      | `"cpu"`                 |
| `--no-fps`              | 隐藏窗口顶部的 FPS 显示                                    | 默认显示                |
| `--show-conf`           | 在对象标签上显示置信度分数                                 | 默认不显示              |
| `--track-len`           | 轨迹历史最大保留帧数                                       | `30`                    |

### 运行示例

```python
# 使用默认摄像头运行（CPU 推理）
python main.py

# 处理视频文件，保存结果，并显示置信度
python main.py --source demo.mp4 --save-video --output result.avi --show-conf

# 使用 GPU 加速，提高检测精度
python main.py --source 0 --device cuda:0 --conf 0.5
```

## 交互操作指南

| 按键 / 操作       | 功能描述                                           |
| :---------------- | :------------------------------------------------- |
| **鼠标左键点击**  | 选中画面中的目标（显示 ID、类别、距离）            |
| **Q**             | 退出程序                                           |
| **P**             | 暂停 / 继续播放视频                                |
| **C**             | 清除当前选中的目标                                 |
| **R**             | 清除所有历史轨迹线                                 |
| **S**             | 截取当前画面并保存为 PNG 文件                      |
| **K**             | 进入 / 退出 **标定模式**（用于设置距离估算参考值） |

### 距离标定说明

程序通过像素高度反比估算距离，使用前需先标定参考物：

1. 按 **K** 键进入标定模式（窗口右上角出现 `CALIBRATION MODE` 提示）。
2. 鼠标点击一个**已知实际距离**的物体（例如一个站立的人）。
3. 在终端输入该物体到摄像头的实际距离（单位：米）并回车。
4. 标定完成后，选中同类物体时即可显示估算距离。

**注意**
>如果想实现对各个物品的识别，请将模型更改为yolo26n.pt，不过此时仅能识别、计数、轨迹和测距，不能进行骨架绘制。
>如果想要进行人体骨架绘制请用yolo26n-pose.pt，不过此时仅能对人实现识别、计数、轨迹、测距和人体骨架绘制。
